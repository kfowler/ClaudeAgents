# Hands-On Workshop Catalog: Claude Code 2.0

**Interactive Learning Experiences Designed for All Learning Styles and Abilities**

## Workshop Philosophy

Our hands-on workshops follow universal design principles to ensure every participant can engage meaningfully regardless of technical background, learning preferences, or accessibility needs. Each workshop combines theory with immediate practical application in a supportive, inclusive environment.

## Accessibility-First Workshop Design

### Universal Participation Features
- **Multiple Interaction Methods**: Keyboard, mouse, touch, voice, and assistive technology support
- **Flexible Pacing**: Self-paced modules with optional instructor synchronization points
- **Multiple Communication Channels**: Text chat, voice, sign language interpretation available
- **Collaborative Tools**: Accessible screen sharing, collaborative editing, and group coordination
- **Error-Tolerant Environment**: Safe practice spaces where mistakes are learning opportunities

### Accommodation Support
- **Screen Reader Optimization**: All workshop materials and interactive elements fully compatible
- **Keyboard Navigation**: Complete workshop functionality available via keyboard shortcuts
- **Visual Accessibility**: High contrast modes, scalable text, color-blind friendly design
- **Motor Accessibility**: Extended interaction times, alternative input methods, voice control
- **Cognitive Support**: Clear instructions, progress tracking, memory aids, simplified language options

## Workshop Categories

### Beginner Workshops (No Prior Experience Required)

#### WS-101: "Your First AI-Enhanced Project" 
**Duration**: 2 hours  
**Format**: Instructor-led with breakout exercises  
**Max Participants**: 12 (optimal for individual attention)  
**Accessibility Features**: Full screen reader support, keyboard navigation, extended time options

**Workshop Overview**:
Participants create their first complete project using Claude Code 2.0's AI features. Starting with a simple portfolio website, learners experience the full development lifecycle from natural language request through deployment.

**Learning Objectives**:
- Make successful AI-enhanced requests using natural language
- Understand how AI selects and coordinates agents
- Complete a full project from conception to deployment
- Configure basic accessibility and personalization settings

**Workshop Structure**:
```
Hour 1: Foundation and Setup
├── 0:00-0:15 - Welcome and accessibility setup
├── 0:15-0:30 - Claude Code 2.0 interface tour
├── 0:30-0:45 - First AI request walkthrough
└── 0:45-1:00 - Hands-on practice with guidance

Hour 2: Project Development  
├── 1:00-1:15 - Portfolio project planning
├── 1:15-1:45 - Guided development with AI assistance
└── 1:45-2:00 - Deployment and celebration
```

**Accessibility Accommodations**:
- **Screen Reader Users**: Pre-workshop screen reader optimization session (15 minutes)
- **Keyboard Users**: Keyboard shortcut reference card and practice time
- **Cognitive Support**: Written step-by-step guides with checkboxes for progress tracking
- **Motor Impairments**: Extended interaction times and voice control setup assistance
- **Visual Impairments**: High contrast setup and screen magnification support

**Materials Provided**:
- Pre-configured development environment
- Step-by-step written guide with screenshots and alt-text
- Audio recording of all instructions for later reference
- Keyboard shortcut reference card
- Emergency help contact information
